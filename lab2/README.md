# Lab 2: Data Structures in Python

This lab focuses on Python data structures including lists, tuples, dictionaries, and sets. It covers manipulation, analysis, and organization of data using these fundamental structures.

## Topics Covered
- Lists and list operations
- Tuples and immutable sequences
- Dictionaries and key-value pairs
- Sets and unique elements
- Data manipulation and analysis
- Sorting and filtering
- Aggregation operations
