# Lab 3: File Handling & Exception Handling

This lab covers comprehensive file handling operations and exception handling for robust file operations with different file formats including text files, CSV files, and JSON files.

## Topics Covered
- File read and write operations
- Exception handling (try-except blocks)
- Working with different file formats (TXT, CSV, JSON)
- File error handling
- Data validation during file operations
- Menu-driven file programs
